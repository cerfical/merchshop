# merchshop

## Использование

Чтобы произвести минимальный запуск приложения можно использовать `docker compose`:

```
docker compose up
```

Для упрощения процесса разработки, используются настройки по умолчанию, а контейнер БД запускается с `POSTGRES_HOST_AUTH_METHOD=trust`, потому установка пароля или каких-либо дополнительных конфигураций не требуется.
